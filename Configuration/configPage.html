Channel Management

Use the Jellyfin API or create custom channels through the CableCast API endpoints:
  • GET /api/cablecast/channels - List all channels
  • POST /api/cablecast/channels - Create a new channel
  • PUT /api/cablecast/channels/{id} - Update a channel
  • DELETE /api/cablecast/channels/{id} - Delete a channel
  • GET /api/cablecast/channels/{id}/current - Get current program
  • GET /api/cablecast/channels/{id}/schedule - Get channel schedule
